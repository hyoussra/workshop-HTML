HTML 5

HTMLou *HyperText Markup Language*, est le langage de base utilisé pour structurer et présenter le contenu des pages Web. Il se compose de balises entourées de crochets qui définissent différents types d'éléments tels que les titres, les paragraphes, les images et les liens. Par exemple, indique un titre principal, tandis que crée un paragraphe de texte. Chaque page HTML commence avec une déclaration , suivie de la balise racine qui contient les balises (pour les métadonnées comme le titre de la page) et body (pour le contenu visible). HTML constitue la structure d'une page, qui peut ensuite être stylisée avec CSS et rendue interactive avec JavaScript.
